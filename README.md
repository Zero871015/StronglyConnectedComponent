# StronglyConnectedComponent

## DOTO

* Get strongly connected component(SCC) given a directed graph.

## Algorithm

1. Use DFS get finish time for erery vertex in G.
2. Get Transpose(G).
3. Use DFS on G, order by finish we get from 1.
